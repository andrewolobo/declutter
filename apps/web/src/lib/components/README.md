# Component Organization

## Directory Structure

### `/common`

Reusable UI components used throughout the application:

- Button
- Input
- Card
- Modal
- LoadingSpinner
- Avatar
- Badge
- Toast

### `/layout`

Layout components for page structure:

- Header
- Footer
- Sidebar
- Navigation

### `/features`

Feature-specific components:

- Post components (PostCard, PostForm, PostDetail)
- User components (UserProfile, UserAvatar)
- Feed components (FeedList, FeedItem)
- Message components
- Category components
